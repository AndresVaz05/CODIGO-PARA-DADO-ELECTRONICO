# 🎲 Dado Electrónico con Arduino

Proyecto académico para la materia **Electrónica Básica**.  
Implementación de un dado electrónico de seis caras utilizando un microcontrolador Arduino Uno, una matriz de 7 LEDs y un pulsador. El sistema genera un número aleatorio al presionar el botón y lo visualiza mediante el patrón de puntos característico de un dado convencional.

---

## 📌 Características

- ✅ Generación de números aleatorios entre 1 y 6.
- ✅ Representación visual mediante **7 LEDs** configurados en forma de dado.
- ✅ Control mediante **pulsador** con antirrebote (debounce) por software.
- ✅ Diseño validado en **Tinkercad**, prototipado en **protoboard** y ensamblaje final en **placa de baquelita**.
- ✅ Código optimizado y comentado para fácil comprensión.

---

## 🛠️ Tecnologías y componentes

| Componente | Cantidad | Descripción |
|------------|----------|-------------|
| Arduino Uno R3 | 1 | Microcontrolador principal |
| LEDs rojos | 7 | Indicadores visuales |
| Resistencias 330 Ω | 7 | Limitación de corriente para LEDs |
| Resistencias 1 kΩ | 1 | Pull‑down para el pulsador |
| Pulsador de 4 pines | 1 | Entrada de usuario |
| Protoboard | 1 | Pruebas temporales |
| Placa de baquelita | 1 | Ensamblaje permanente |
| Cables jumper | varios | Conexiones eléctricas |

### Software
- **Tinkercad Circuits** – Simulación y validación.
- **Arduino IDE** – Programación y carga del código.
